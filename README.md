## Universe Launcher ğŸš€

This repository includes a powerful universe launcher system that creates and launches universes, galaxies, and solar systems. The Phu Quoc Nguyen Universe Launcher demonstrates cosmic-scale creation capabilities.

### Features

- **Universe Creation**: Create universes with fundamental physics constants
- **Galaxy Generation**: Generate spiral, elliptical, and irregular galaxies
- **Solar System Builder**: Build solar systems with planets, moons, and asteroid belts
- **Integrated Launcher**: Launch everything with a single command

### Installation

```bash
# No dependencies needed - pure Node.js!
node --version  # Requires Node.js 12+
```

### Usage

Launch all universes, galaxies, and solar systems:

```bash
# Run the main launcher
node index.js

# Or use npm scripts
npm start           # Launch the main entry point
npm run launch      # Launch all components
npm test           # Run all tests
```

Run individual components:

```bash
# Create a universe
node create-universe.js

# Create galaxies
node create-galaxy.js

# Create solar systems
node create-solar-system.js
```

### What Gets Created

When you run the launcher, it creates:

- **1 Universe**: The Phu Quoc Universe (13.8 billion years old, 93 billion light years in diameter)
- **5 Galaxies**: 
  - Phu Andromeda (spiral, 220,000 light years)
  - Phu Milky Way (spiral, 100,000 light years)
  - Phu Triangulum (spiral, 60,000 light years)
  - Phu Centaurus (elliptical, 180,000 light years)
  - Phu Magellan (irregular, 14,000 light years)
- **5 Solar Systems**: Each with 8 planets, multiple moons, and asteroid belts
