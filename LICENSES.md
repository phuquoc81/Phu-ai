## 6. License Compliance

### How We Ensure License Compliance

1. **Dependency Review**: All new dependencies are reviewed for license compatibility before addition
2. **Automated Scanning**: Dependabot and security workflows check for license issues
3. **Attribution Maintenance**: This file is updated when new dependencies are added
4. **License Compatibility**: We only use licenses compatible with Apache 2.0
